Design Discussion Paragraph

The operator terminal is cohesive with the member and provider records because they work together to perform the add, update, and remove member/provider tasks. The manager terminal is cohesive with the report because they work together to create and send out the report to members and providers. The timer is also cohesive with the report as they work together to perform the main accounting procedure, and also send out the report to members and providers. The provider terminal is cohesive with the service report as they work together to create the service report. The provider is also cohesive with the provider directory as they perform the fulfill provider directory request task. There is coupling between the provider, provider directory, and service report but we believe this coupling to be necessary. There is also coupling between the manager terminal, timer, and report but this coupling is also necessary. Overall our design demonstrates high cohesion and low coupling which make for an effective design.
